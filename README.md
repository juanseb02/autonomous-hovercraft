# Autonomous Hovercraft Platform

## Overview
This project implements a small autonomous hovercraft capable of navigating a structured environment using onboard sensors and embedded control logic.

The system integrates mechanical design, electronics, and embedded programming to create a functioning robotic platform.

## System Architecture
The hovercraft uses a microcontroller to process sensor inputs and control propulsion and steering in real time.

Components include:
- propulsion motor
- steering mechanism
- obstacle detection sensors
- embedded control system

## Features
- real-time obstacle detection
- embedded control system
- hardware–software integration
- navigation testing on custom track

## Technologies
- C/C++
- Embedded Systems
- Microcontrollers
- Robotics
- Sensor Integration

## Testing
The hovercraft was tested on a custom-built obstacle course to evaluate navigation performance and system behavior.

## Future Improvements
- improved navigation algorithms
- additional sensors
- closed-loop control
